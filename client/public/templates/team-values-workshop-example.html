🎯 Team Values Workshop

Example completed workshop for a Chick-fil-A team

📋 Workshop Facilitation Guide (30 minutes)

  • Gather your team in a comfortable setting away from distractions
  • Explain that you want to identify shared values that guide how you work together
  • Start with values identification (10 minutes) - ask "What matters most to us as a team?"
  • Use Start, Stop, Continue framework (15 minutes) to identify specific behaviors
  • Create action plan (5 minutes) and assign someone to create visual reminders

📋 Workshop Information
